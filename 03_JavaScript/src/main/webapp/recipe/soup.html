국

  • 미역국

    1. 미역은 물 500ml(약 2.5컵)에 10분간 불려주세요.

    2. 미역은 체를 이용해 건져주세요. 이때 미역 불린 물은 버리지 않고 미역국 끓일 때 물 대신 사용해주세요.

    3. 소고기 겉면이 절반 정도 익을 때쯤 국간장 1큰술을 넣어 볶아주세요.

    4. 쌀뜨물 혹은 미역 불린 물 400mL(약 2컵)를 붓고 강불로 올려주세요.

    5. 국물이 끓기 시작하면 물 1L(약 5컵)를 넣어주세요. 다시 끓어오르면 중약불로 줄인 뒤 15분간 끓여주세요.

    6.국간장 1큰술을 넣고 부족한 간은 소금으로 더해주세요. 엄마 손맛 가득한 미역국 완성입니다.

  • 콩나물국

    1. 콩나물은 머리 껍질 떼어내고 꼬리부분 더러우면 떼어내서 손질한 후 깨끗이 씻어 준비해 주세요.

    2. 냄비에 물 4국그릇 부어준 후, 소금 2작은술 넣고 팔팔 끓여줍니다.

    3. 물이 팔팔 끓어 오르면 손질해 두었던 콩나물을 넣고 뚜껑을 덮은 후, 5분동안 그대로 끓여 주세요!

    4. 5분 후 뚜껑을 열고, 다진마늘 1큰술, 파 1줌 넉넉히 넣고 고추가루 1큰술 넣고 조금 더 끓여 주면 완성입니다.

  • 소고기무국

    1. 소고기는 찬물에 10분정도 담가 핏물을 제거해주세요.

    2. 무는 네모지게 토막 , 대파 1대는 큼직큼직 어슷 썰어주세요.

    3. 참기름 1큰술을 넣어준 후 고기를 넣고 겉면의 색이 변할 때까지 볶아주세요.

    4. 고기가 갈색으로 변하면 무를 넣어 준 후 살짝 투명해질때까지 볶아주세요.

    5. 무와 고기가 잘 볶아지면 물을 넣고 중불로 끓여주세요.

    6. 국간장 2큰술, 소금 1작은술, 다진마늘 1/2큰술, 설탕1/2큰술을 넣어 간을 해주세요.

    7. 중불로 20분간 보글보글 끓여준 후 대파와 후추를 톡톡 뿌려주면 완성!!
