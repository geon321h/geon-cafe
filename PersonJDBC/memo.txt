drop sequence perseq;
create sequence perseq
increment by 1
start with 1
minvalue 1
;

drop table person;
create table person(
  num number PRIMARY KEY,
  name VARCHAR2(10),
  age NUMBER,
  gender VARCHAR2(10),
  birth date DEFAULT sysdate
);

DAO(Data Access Object)
- DB를 사용해 데이터를 조회하거나 조작하는 기능을 전담하는 오브제트
- 저장소에 테이터를 입력 , 조획, 수정, 삭제 들의 처리를 담당

DTO(Data Transfer Object)
- bean, VO(Value Object)라고 부르기도 한다.
- 데이터의 전달을 위해 만들어진 오브젝트
- 데이터를 담을 private 변수와 getter,setter 메서드로 구성된다.
